# SearchLight

Этот РИДМИ также доступен на [English](README.md)

SearchLight — это инструмент командной строки (CLI), предназначенный для управления и взаимодействия с кластерами OpenSearch. Он предоставляет структурированный и расширяемый способ выполнения административных и операционных задач, таких как управление безопасностью, работа с политиками индексов и мониторинг кластера.

Планируется как OpenSearch-SwissKnife в мире API OpenSearch.

Проект построен с использованием `typer` для реализации CLI и использует официальный клиент `opensearch-py`.

## Возможности

- Управление безопасностью (пользователи, роли)
- Управление политиками индексов
- Просмотр и контроль задач кластера
- Получение информации о нодах
- Единый CLI-интерфейс для работы с OpenSearch
- Модульная и расширяемая архитектура

## Архитектура

Проект реализован с использованием модульного подхода с разделением ответственности между CLI, бизнес-логикой и инфраструктурой:

```
searchlight/
├── main.py              # Точка входа CLI
├── cli.py               # Определение команд
├── client.py            # Инициализация клиента OpenSearch
├── core/                # Базовые операции (например, backup, restore)
├── services/            # Слой взаимодействия с OpenSearch API
│   ├── security.py
│   ├── index_management.py
│   ├── tasks.py
│   ├── nodes.py
│   └── templates.py
├── utils/               # Утилиты (логирование, форматирование, вспомогательные функции)
```

### Обзор компонентов

- **CLI-слой** — определяет команды и аргументы с использованием Typer  
- **Слой сервисов** — инкапсулирует работу с OpenSearch API  
- **Слой клиента** — отвечает за настройку подключения  
- **Core-слой** — реализует высокоуровневые операции  
- **Utils** — общие утилиты, такие как логирование и форматирование  

## Установка

Клонирование репозитория:

```bash
git clone https://github.com/DuraCHYo/SearchLight.git
cd SearchLight
```

Установка зависимостей через Poetry:

```bash
poetry install
```

Альтернативно через pip:

```bash
pip install .
```

## Конфигурация

SearchLight поддерживает конфигурацию через аргументы командной строки и переменные окружения.

Основные параметры:

- `--host` (или `OS_HOST`)
- `--port` (или `OS_PORT`)
- `--auth` (формат: `username:password`)
- параметры SSL

Пример:

```bash
searchlight --host localhost --port 9200 --auth admin:admin
```

## Использование

### Операции безопасности

```bash
searchlight security ...
```

Управление пользователями и ролями в кластере OpenSearch.

### Политики индексов

```bash
searchlight policies ...
```

Создание, обновление и просмотр политик управления индексами.

### Задачи кластера

```bash
searchlight tasks ...
```

Просмотр и управление выполняющимися задачами.

### Ноды кластера

```bash
searchlight nodes ...
```

Получение информации о нодах кластера.
