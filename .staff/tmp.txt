# ⚡️ Шпаргалка по uv (вместо Poetry)

### 🚀 Запуск кода
Вместо `poetry run` используйте `uv run`:
```bash
uv run searchlight/main.py --host localhost --port 9200 --auth 'admin:myStrongPassword123!' --disable-verify-ssl security list-users
```

---

### 📂 Виртуальные окружения
**uv** по умолчанию создает папку `.venv` прямо в корне проекта.

*   **Локация:** `./.venv`
*   **Автоматизация:** `uv` сам создаст окружение при первой команде `uv run` или `uv sync`.

---

### 📝 Основные команды

#### 1. Начало работы
*   `uv init` — создать новый проект (аналог `poetry init/new`).
*   `uv sync` — установить все зависимости и создать `.venv` (аналог `poetry install`).

#### 2. Управление библиотеками
*   `uv add <package>` — добавить библиотеку.
*   `uv add --dev <package>` — добавить библиотеку для разработки.
*   `uv remove <package>` — удалить библиотеку.
*   `uv lock --upgrade` — обновить зависимости (аналог `poetry update`).

#### 3. Запуск и работа
*   `uv run <command>` — запустить скрипт внутри окружения.
*   `uv tree` — дерево зависимостей.
*   `uv python install 3.12` — установить нужную версию Python.
